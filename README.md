# Tic-Tac-Toe (React)

A simple Tic-Tac-Toe game built with **React** and **Vite**. Players can edit their names, take turns, view the move log, and restart the game.

## Features

* 3Ã—3 game board
* Player name editing
* Active player highlight
* Winner + draw detection
* Restart button
* History log
